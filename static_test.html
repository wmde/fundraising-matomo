Adding tracking parameters

You can add campaign and keyword parameters to the URL. In a default Matomo installation, the parameter names are mtm_campaign and mtm_keyword. You can look up other allowed parameter names (like piwik_campaign, pk_campaign and even Google-Analytics-style utm_campaign) in the file global.ini.php of the Matomo configuration Look for the config keys campaign_var_name and keyword_var_name.
