Matomo "Impression Generation" test page

This file logs impressions to Matomo.
Go to http://localhost:8090 to see the Matomo dashboard and check if the impressions come in.

Adding tracking parameters

You can add campaign and keyword parameters to the URL. In a default Matomo 4 installation, the parameter names are `mtm_campaign` and `mtm_keyword`. You can look up other allowed parameter names (like `piwik_campaign`, `pk_campaign` and even Google-Analytics-style `utm_campaign`) in the file `global.ini.php` of the Matomo configuration.
